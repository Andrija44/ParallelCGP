# ParallelCGP
Završni rad na FER-u u akademskoj godini 2024/2025

Main grana označava sekvencijalnu verziju programa

## Pokretanje
### Unix
g++ 13.3.0

`cmake -S .. -B . -DCMAKE_CXX_COMPILER=g++`

`make`
### Windows
g++ 11.2.0

`cmake -S .. -B . -DCMAKE_CXX_COMPILER=g++ -G "MinGW Makefiles"`

`make`
